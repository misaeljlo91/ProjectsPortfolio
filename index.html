Sobre mí

Me considero una persona muy curiosa por tanto soy un autodidacta por naturaleza, me gusta leer e investigar. Aprender nuevas cosas ha sido mi proyección personal.

Me formé como ingeniero en Venezuela, luego de una intensa búsqueda laboral en mi profesión se me presentó la oportunidad para trabajar en el área de marketing de un instituto permitiéndome desarrollar nuevas habilidades en un campo completamente nuevo para mí en ese entonces. Luego de esa experiencia mi interés hacía el estudio de tecnologías fue creciendo lo que me llevó a introducirme en el mundo IT, realizando un Máster en Marketing Digital y Analítica Web, haciendo uso de WordPress y creando mi propio blog.

Un día se me presentó la oportunidad en MindHub para hacer uno de sus bootcamps y esto me permitió inclinarme hacia el Desarrollo Web Front-End (HTML, CSS, JavaScript, Vue.js, Bootstrap), adquiriendo habilidades y capacidades también en el Desarrollo Back-End (JAVA, Spring Boot, Spring Security, Gradle, PostgreSQL).

Me gustaría especializarme en el Diseño UX/UI pero no me cierro a la oportunidad de aprender nuevas herramientas y seguir formándome como Desarrollador Web Full Stack. Sé que un día podré superar todas las expectativa que tengo para mí y mi futuro.

TGiF - ProPublic

Este proyecto se realizó por solicitud de la academía para completar el ciclo de enseñanaza Front-End. En el mismo se trabajó con una API pública de los miembros del senado y del congreso de Estados Unidos de América en tiempo real.

Para este proyecto se utilizó HTML5 para su desarrollo y muestra de la información; en principio se usó CSS vanilla y posteriormente se usó el framework de Bootstrap para un mejor diseño. Del mismo modo para darle funcionalidad al proyecto se usó un archivo JavaScript que en principio, al igual que para el estilo, se uso de manera vanilla y posteriormente le fue agregado en framework de Vue.js. Estás herramientas facilitaron la elaboración del proyecto y el mismo fue presentado como requisito para la aprobación del módulo.

Ball D'or

Este proyecto comprendió realizar una tienda online y que a su vez fuera conectada con la plataforma de banco que se desarrolló anteriormente. Fue un proyecto grupal donde participamos cinco (05) personas, cada uno con un rol dentro del proyecto y, de libre decisión en cuanto a su contenido pero cumpliendo con las pautas que establecieron para su desarrollo.

El mismo fue desarrollado en un tiempo corto y aplicando los conocimientos de Back-End y Front-End que adquirimos durante nuestra formación en el bootcamp.

En este proyecto se utilizó la herramienta de Git y la plataforma de GitHub para pushear cada uno de los cambios y modificaciones que se iban realizando a lo largo de su desarrollo, se utilizó Java como lenguaje para el desarrollo del Back-End y donde se hicieron modelos y controladres para ser usados en el Front-End; en cuanto al desarrollado de la vista al usuario se utilizó HTML5, CSS, JavaScript, Bootstrap y Vue.js.

Mi rol dentro del equipo fue el desarrollo visual o la parte Front-End, de igual forma para añadir ciertas funcionalidades que se implementarón tuve que desarrollar servicios de API Rest para dichas funcionalidades, entre ella la opción de Olvidó su contraseña, cambio de datos del usuario y cambio de contraseña.
